# 排版

规范化文档排版，解决中英文混排、标点、段落等格式问题。

## 是什么

按照一套完整的排版规则，对文档进行规范化处理，包括：
- 段落结构调整
- 中英文/中数字空格
- 标点符号规范
- 专有名词大小写
- 链接格式

## 触发指令

```
排版 [文件名]
```

例如：`排版 从高三到大学，我失去了什么.md`

## 核心规则

### 1. 段落结构
- 每段之间空一行，不缩进
- 一段 3~5 行，超过就拆

### 2. 空格（重点）
- 中文 + 英文：Lean Cloud → Lean Cloud
- 中文 + 数字：2025年 → 2025 年
- 数字 + 单位：10Mbps → 10 Mbps
- 中文 + 破折号：重点—— → 重点 ——

### 3. 标点符号
- 中文用全角：「」
- 破折号前后空格：这是重点 —— 必看
- 省略号用……

### 4. 专有名词
- GitHub（不是 github）
- 豆瓣FM（不是 豆瓣 fm）
- JavaScript（不是 Js）

### 5. 链接
- 前后加空格
- 加粗处理

## 输出

返回修改的具体内容清单。
