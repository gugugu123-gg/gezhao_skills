# 思想编码器

将深层思想压缩为高传播力的短内容。

## 是什么

一个帮助你把"脑子里高维思想"拍扁、压缩成短句的技能。

解决的核心问题：**为什么绝大多数聪明人发帖水平都很 Mid？**

因为真正的难点不在于"产生好想法"，而在于"有损编码"——如何把思想压缩进寥寥数语，还能让读者准确还原。

## 触发指令

```
压缩 [你的想法/观点/文章片段]
```

支持：
- `压缩 xxx`
- `思想压缩 xxx`
- `精简 xxx`
- `压缩观点 xxx`

## 输出

2 条推文：
- **锐利版** - 最大冲击，像子弹打进脑子
- **通透版** - 照亮盲区，让人"原来如此"

字数：≤60 字

## 核心原则

1. **穿透 > 压缩** - 找到思想的生成机制，不是复述结论
2. **陌生具体 > 熟悉具体** - 用读者没想到的例子
3. **结构即力量** - 前半句建预期，后半句打破它
4. **解压种子** - 让没读过原文的人产生正确推论

## 示例

输入：
```
很多人以为发帖的难点在于"产生好想法"，其实不然。真正的难点在于表达——如何将一个高维的思想拍扁，压缩进寥寥数语中，并确保它能在另一个人的脑海里被准确地重构还原。
```

输出：
```
【锐利版】
发帖最大的坑：以为难点是"想法不够"，其实99%输在"表达太损"。

【通透版】
你脑子里那东西很高级，但别人看到的只是一堆碎渣——不是他们笨，是你没包好。
```

## 使用场景

- 写完文章后，生成推广文案
- 把深度观点转成社交媒体短内容
- 学习如何"穿透"表达而非"复述"
