# gezhao_skills
这是我个人使用的一些 Skills 配置与工具集合。

这些 Skills 主要用于：

- 提高 AI 使用效率
- 结构化思考与问题拆解
- 自动化工作流程
- 提升学习与写作效率

---

# 使用说明

⚠️ 注意：

本仓库中的部分 Skill 使用了 **本地路径** 或 **特定文件结构**，  
在使用前，你需要根据自己的实际环境进行修改。

需要重点检查的内容包括：

- 文件路径（File Path）
- 本地目录结构（Folder Structure）
- 依赖文件位置（Dependencies）
- 配置文件路径（Config Path）

例如：

```bash
/path/to/my/local/folder
```

# skills

## 质询与审视

| 技能                    | 说明                                                         |
| ----------------------- | ------------------------------------------------------------ |
| gap-detector            | 找盲区。轻量级质询入口，发现想法、观点、文章、任务、面试准备里的盲区，生成高质量追问问题 |
| 质询复盘                | 高标准非迎合的深度复盘。对决策、观点、方法、经历、外部材料进行逻辑审查，输出可沉淀的复盘报告 |
| path-compliance-checker | AI 协作路径合规检测。判断任务是否结构完整，决定直接执行、追问补全、还是阻断澄清 |

## 学习与提炼

| 技能                             | 说明                                                         |
| -------------------------------- | ------------------------------------------------------------ |
| youtube-video-learning-assistant | YouTube 视频内容学习助手。把视频字幕整理成结构化笔记，提炼方法论，沉淀为个人知识资产 |
| youtube-video-benchmark-analysis | YouTube 视频对标拆解。分析内容为什么能火、结构和表达好在哪里、哪些可以迁移到自己的创作 |
| insight-extractor                | 外部内容认知提炼。把博客、文章、播客、字幕等整理成结构化笔记，输出 Summary、Takeaways、方法论、刁钻提问 |

## 创作与表达

| 技能                        | 说明                                                         |
| --------------------------- | ------------------------------------------------------------ |
| gezhao-writing-style        | 按照个人写作风格改写文章。支持口播版（1-3行/段）和文章版（3-5行/段） |
| writing-style-skill-creator | 写作风格 Skill 创建器。通过"分析→仿写→修改→迭代"四步，打造专属 AI 写作风格规则 |
| 思想编码器                  | 将深层思想压缩为高传播力的短内容。输出两条推文：锐利版（最大冲击感）和通透版（照亮盲区） |
| 排版                        | 规范化文档排版。解决中英文混排、标点、段落等格式问题         |

## 个人成长

| 技能                     | 说明                                                         |
| ------------------------ | ------------------------------------------------------------ |
| 自我对话                 | 通过持续提问挖掘内心深处的声音。探索兴趣、需求、潜能和潜意识 |
| interactive-study-skill  | 基于 Bloom 2 Sigma + Mastery Learning 的互动学习工作流。生成序列化笔记，评估理解后再推进 |
| obsidian-draft-organizer | 自动整理 Obsidian 草稿文件夹。读取、分类打标、移动到分类目录、创建整合文档 |
